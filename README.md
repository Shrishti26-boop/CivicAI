# 🚀 CivicAI - AI Powered Smart Civic Complaint Management System

## 📌 Overview

CivicAI is an AI-powered civic complaint management system that enables citizens to report public issues such as potholes, garbage, water leakage, and streetlight failures. The platform uses AI to classify uploaded complaint images, captures the user's live location, and helps government officers efficiently track and resolve issues.

---

## ✨ Features

* 🤖 AI-based image classification using TensorFlow
* 📷 Upload complaint images
* 📍 Live location detection
* 🗺️ Interactive map using Leaflet & OpenStreetMap
* 🔥 Firebase Firestore database
* ☁️ Firebase Storage integration
* 👤 Citizen & Officer Login
* 📋 Complaint tracking
* 🔔 Notification system
* 🏆 Reward points & badges
* 🥇 Leaderboard
* 📊 Analytics Dashboard

## 🚀 How to Run

```bash
git clone https://github.com/Shrishti26-boop/CivicAI.git
cd CivicAI

python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

pip install -r requirements.txt

python app.py
```
